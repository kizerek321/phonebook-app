# Project Overview

This project is a phonebook application that uses Google GenAI to interact with the phonebook.

## Features

- Add contacts
- Delete contacts
- Search contacts
- List contacts

## Tech Stack

- Python
- FastAPI
- SQLAlchemy
- Google GenAI

## Installation

```bash
cd backend
pip install -r requirements.txt
```

## Usage

```bash
uvicorn main:app --reload
```

## API Documentation

```bash
http://localhost:8000/docs
```
